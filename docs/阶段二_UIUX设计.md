# 交付物二：设计风格定义（P1-02）

## 2.1 设计理念

**关键词**：温暖 · 活力 · 亲切 · 全龄友好

> HomeFun 的视觉风格应传递"家庭温馨"与"欢乐活力"的双重感受。色彩以暖色调为主，搭配圆润的图形语言，让从 8 岁孩子到 70 岁祖辈都能感到舒适和亲切。

## 2.2 色彩系统

### 2.2.1 主色板（Brand Colors）

| 色彩角色 | 色值 | 用途 | 说明 |
|---------|------|------|------|
| **主色 Primary** | `#FF6B35`（暖橙色） | 主按钮、Tab 高亮、关键操作 | 温暖活力，代表家庭欢乐 |
| **主色浅 Primary Light** | `#FFE8DD` | 主色背景、选中态背景 | 柔和的橙色底色 |
| **主色深 Primary Dark** | `#E55A25` | 按钮按下态、文字强调 | 深橙色，增加层次 |
| **辅助色 Secondary** | `#4ECDC4`（薄荷绿） | 成功状态、游戏元素、辅助按钮 | 清新自然，代表轻松愉悦 |
| **辅助色浅 Secondary Light** | `#E0F7F5` | 辅助背景 | 柔和的薄荷绿底色 |
| **强调色 Accent** | `#FFD93D`（明亮黄） | 积分、成就、高亮标签 | 欢快明亮，代表奖励和成就 |
| **游戏色 Game** | `#6C5CE7`（梦幻紫） | 游戏模块专属色 | 梦幻有趣，激发游戏兴趣 |
| **影音色 Media** | `#FF6B81`（珊瑚粉） | 影音模块专属色 | 温馨浪漫，适合影音氛围 |
| **规划色 Plan** | `#00B894`（翡翠绿） | 规划模块专属色 | 有序安定，代表规划感 |

### 2.2.2 语义色（Semantic Colors）

| 色彩角色 | 色值 | 用途 |
|---------|------|------|
| **成功 Success** | `#00B894` | 操作成功、在线状态 |
| **警告 Warning** | `#FDCB6E` | 提醒注意、即将超时 |
| **错误 Error** | `#E17055` | 操作失败、表单校验错误 |
| **信息 Info** | `#74B9FF` | 提示信息、帮助说明 |

### 2.2.3 中性色（Neutral Colors）

| 色彩角色 | 色值 | 用途 |
|---------|------|------|
| **标题文字** | `#2D3436` | 一级标题、重要文字 |
| **正文文字** | `#636E72` | 正文内容、描述文字 |
| **辅助文字** | `#B2BEC3` | 占位符、次要信息 |
| **分割线** | `#DFE6E9` | 列表分割、卡片边框 |
| **页面背景** | `#F8F9FA` | 页面底色 |
| **卡片背景** | `#FFFFFF` | 卡片、弹窗背景 |
| **禁用态** | `#DFE6E9` | 禁用按钮、不可操作元素 |

## 2.3 字体系统

### 2.3.1 字体选择

| 用途 | iOS | Android | 说明 |
|------|-----|---------|------|
| **中文** | 苹方（PingFang SC） | 思源黑体（Noto Sans CJK SC） | 系统默认中文字体 |
| **英文/数字** | SF Pro Display | Roboto | 系统默认英文字体 |
| **游戏内文字** | 苹方 Rounded | Noto Sans CJK SC Rounded | 圆润字体增加趣味感 |

### 2.3.2 字号层级

| 层级 | 字号 | 字重 | 行高 | 用途 |
|------|:----:|:----:|:----:|------|
| **H1 大标题** | 28sp | Bold (700) | 36sp | 页面主标题 |
| **H2 标题** | 22sp | Bold (700) | 30sp | 模块标题 |
| **H3 小标题** | 18sp | SemiBold (600) | 26sp | 卡片标题、区域标题 |
| **H4 子标题** | 16sp | Medium (500) | 24sp | 列表标题、按钮文字 |
| **Body 正文** | 15sp | Regular (400) | 22sp | 正文内容 |
| **Body 小字** | 13sp | Regular (400) | 20sp | 辅助说明、标签 |
| **Caption 注释** | 12sp | Regular (400) | 18sp | 时间戳、极小文字 |
| **长辈模式** | 各层级 +4sp | — | — | 祖辈用户专用 |
| **儿童模式** | 各层级 +2sp | — | — | 儿童用户专用 |

## 2.4 圆角系统

| 层级 | 圆角值 | 用途 |
|:----:|:-----:|------|
| **无圆角** | 0dp | 分割线、全宽元素 |
| **小圆角** | 8dp | 小按钮、标签、输入框 |
| **中圆角** | 12dp | 卡片、弹窗、列表项 |
| **大圆角** | 16dp | 大卡片、底部弹窗 |
| **全圆角** | 999dp | 头像、胶囊按钮、Badge |
| **超级圆角** | 24dp | 游戏卡片、特殊容器 |

> **设计原则**：HomeFun 整体偏向圆润风格，默认使用中圆角（12dp），营造亲切友好的视觉感受。

## 2.5 间距系统

采用 **4dp 基准网格**，所有间距为 4 的倍数。

| Token 名称 | 值 | 用途 |
|-----------|:--:|------|
| `space-xs` | 4dp | 图标与文字间距、紧凑元素内间距 |
| `space-sm` | 8dp | 列表项内间距、小元素外边距 |
| `space-md` | 12dp | 卡片内间距、段落间距 |
| `space-base` | 16dp | 页面左右边距、标准间距 |
| `space-lg` | 20dp | 模块间距、大卡片内间距 |
| `space-xl` | 24dp | 区块间距 |
| `space-2xl` | 32dp | 大区块间距 |
| `space-3xl` | 48dp | 页面顶部/底部留白 |

## 2.6 阴影系统

| 层级 | 阴影参数 | 用途 |
|:----:|---------|------|
| **Shadow-sm** | `0 1dp 3dp rgba(0,0,0,0.08)` | 轻微浮起、卡片 |
| **Shadow-md** | `0 2dp 8dp rgba(0,0,0,0.12)` | 浮动按钮、弹窗 |
| **Shadow-lg** | `0 4dp 16dp rgba(0,0,0,0.16)` | 底部弹窗、模态框 |
| **Shadow-xl** | `0 8dp 32dp rgba(0,0,0,0.20)` | 特殊浮层 |

## 2.7 图标规范

| 属性 | 规范 |
|------|------|
| **图标风格** | 圆角线性图标（Round Line），与整体圆润风格一致 |
| **图标尺寸** | 20dp（导航栏）/ 24dp（列表）/ 32dp（空状态）/ 48dp（功能入口） |
| **图标粗细** | 1.5dp 描边 |
| **图标颜色** | 默认 `#636E72`，激活态跟随模块主题色 |
| **图标库** | 推荐使用 Phosphor Icons（圆角风格）或自定义 SVG |

## 2.8 暗色模式（Dark Mode）

> **MVP 阶段暂不实现**，但设计规范预留暗色模式变量映射：

| 角色 | 亮色 | 暗色 |
|------|------|------|
| 页面背景 | `#F8F9FA` | `#1A1A2E` |
| 卡片背景 | `#FFFFFF` | `#16213E` |
| 标题文字 | `#2D3436` | `#EAEAEA` |
| 正文文字 | `#636E72` | `#A0A0A0` |
| 分割线 | `#DFE6E9` | `#2D2D44` |

# 交付物四：核心页面高保真设计稿（P1-04）

> **说明**：以下为每个核心页面的详细 UI 规格，开发团队可直接据此实现。实际 Figma 设计稿应在此基础上进行视觉细化。

## 4.1 全局框架规格

### 4.1.1 状态栏

| 属性 | 值 |
|------|-----|
| 高度 | 遵循系统安全区域（iOS 44pt / Android 24dp） |
| 背景 | 透明 / 跟随页面背景色 |
| 文字颜色 | 深色页面：白色；浅色页面：`#2D3436` |

### 4.1.2 导航栏（顶部）

| 属性 | 值 |
|------|-----|
| 高度 | 44pt / 48dp |
| 背景 | `#FFFFFF` 或模块主题色 |
| 标题 | H4 层级（16sp Medium） |
| 返回按钮 | 左侧，24dp 图标 |
| 操作按钮 | 右侧，24dp 图标或文字按钮 |

### 4.1.3 底部 Tab 栏

| 属性 | 值 |
|------|-----|
| 高度 | 49pt / 56dp（含安全区域） |
| 背景 | `#FFFFFF`，顶部 1px `#DFE6E9` 分割线 |
| Tab 数量 | 4 个（首页/游戏/影音/规划） |
| 图标尺寸 | 24dp（默认）→ 28dp（选中态） |
| 文字 | Caption 层级（12sp），选中态跟随模块色 |
| 选中态 | 图标 + 文字变为主题色，带微弹动效 |
| 未选中态 | `#B2BEC3` |

### 4.1.4 Tab 模块主题色映射

| Tab | 默认色 | 选中色 | 图标 |
|-----|--------|--------|------|
| 首页 | `#B2BEC3` | `#FF6B35`（主色） | 🏠 Home |
| 游戏 | `#B2BEC3` | `#6C5CE7`（游戏紫） | 🎮 Gamepad |
| 影音 | `#B2BEC3` | `#FF6B81`（珊瑚粉） | 🎬 Film |
| 规划 | `#B2BEC3` | `#00B894`（翡翠绿） | 📅 Calendar |

## 4.2 登录页详细规格

### 页面布局

| 区域 | 规格 |
|------|------|
| 页面背景 | 渐变 `#FFF5F0` → `#FFE8DD`（暖色渐变） |
| Logo 区域 | 距顶部 25% 居中 |
| Logo 图标 | 64dp，品牌橙色 |
| 品牌名 | H1（28sp Bold），`#2D3436` |
| 副标题 | Body（15sp），`#636E72` |
| 表单区域 | 距 Logo 下方 40dp，左右 padding 24dp |
| 输入框 | 高度 48dp，圆角 8dp，背景 `#FFFFFF`，边框 1px `#DFE6E9` |
| 输入框聚焦 | 边框变为主色 `#FF6B35`，阴影 Shadow-sm |
| 登录按钮 | 高度 48dp，圆角 12dp，背景主色 `#FF6B35`，文字白色 H4 |
| 按钮按下态 | 背景 `#E55A25`，缩放 0.98 |
| 微信登录 | 文字按钮，微信绿色 `#07C160`，图标 24dp |
| 底部链接 | Caption（12sp），`#636E72` |

### 输入框状态

| 转场类型 | 动画 | 时长 | 缓动曲线 | 使用场景 |
